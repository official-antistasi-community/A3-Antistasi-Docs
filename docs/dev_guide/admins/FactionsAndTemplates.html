Factions and Template Overview

Please note that this part is still heavily work in progress. For more dev related information please go to our Antistasi-Wiki-for-Devs.

If you have questions like "I want to play against [random faction], how do I do that?" this is the place to look at. Antistasi is categorizing the maps into different climates and based on them as well as the mods loaded you get different Factions with adapted camos and such. In addition there are a few special cases like for example the HIDF (Horizon Island Defense Forces) which you only get on Tanoa.

To make sure you have all the assets loaded, click on the links which direct you to the Steam Workshop Item containing the dependencies.
