How to build Antistasi (Mission)

This guide is depricated! Be aware that this guide only is for Antistasi Version 2.5.x and earlier.

Build with Antistasi Dev Deploy

Build with Antistasi Dev Deploy

The Antistasi Dev Deploy Configurator binaries can be found at Github-Repository. You can find a guide on getting started in the wiki here. Further you can find an information about Configurator over there.

Build with Powershell

Build with Powershell

  1. Download ZIP file from GitHub

  2. Extract ZIP

  3. Right click on PrepareMissionsForPacking.ps and select ‘Open with PowerShell’

  4. The script will create a new folder called PreparedMissions/

  5. Inside this folder you will find multiple subfolders, labeled Antistasi-VersionNumber.MapName

  6. Use a PBO Packager to pack the any of these mission folders
